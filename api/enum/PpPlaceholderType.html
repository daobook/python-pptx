PP_PLACEHOLDER_TYPE¶

Specifies one of the 18 distinct types of placeholder.

Alias: PP_PLACEHOLDER

Example:

from pptx.enum.shapes import PP_PLACEHOLDER

placeholder = slide.placeholders[0]
assert placeholder.type == PP_PLACEHOLDER.TITLE
