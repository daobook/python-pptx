XL_MARKER_STYLE¶

Specifies the marker style for a point or series in a line chart, scatter chart, or radar chart.

Example:

from pptx.enum.chart import XL_MARKER_STYLE

series.marker.style = XL_MARKER_STYLE.CIRCLE
